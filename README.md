# C++ boilerplate

This repository contains a simple boilerplate for C++ projects (under Linux).

It uses CMake, googletest, clang-format and clang-tidy.

## Dependencies 

* make
* cmake 
* gcc/g++ 
* clang-format
* clang-tidy
* boost
* googletest (handled by [CMake-CPM](https://github.com/cpm-cmake/CPM.cmake))
